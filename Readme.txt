shoppingcart.py contains main code with all functional requirements

To run shoppingcart.py : 

1)Open CMD on the particular directory and run "python shoppingcart.py"

2)Enter the input from available products separated with space

3)If unavailable product is entered as input it will be showed as unavailable


test_shoppingcart.py contains necessary testcases of the main program

To run test_shoppingcart.py :

1) Open CMD on the particular directory and run "python -m unittest test_shoppingcart.py"
